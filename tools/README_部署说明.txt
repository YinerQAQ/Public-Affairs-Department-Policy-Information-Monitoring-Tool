========================================
政策信息监控工具 - 部署说明
========================================

【特点】
本版本为「免安装」版本，已内置 Python 运行环境与所有依赖（包括浏览器内核），
拿到文件夹即可使用，无需安装 Python，也无需联网下载依赖。

【本机使用（给同事）】
1. 把整个文件夹复制到目标电脑（任意路径，建议放磁盘根目录或较短路径下）
2. 双击 install.bat 完成首次初始化（仅需运行一次，初始化数据库）
3. 双击 start.bat 启动应用
4. 浏览器打开 http://127.0.0.1:5000

【服务器部署（后台运行 + 开机自启）】
1. 把文件夹复制到服务器
2. 右键 deploy_server.bat → 以管理员身份运行
3. 部署完成后，其他电脑通过 http://服务器IP:5000 访问
4. 服务器重启后会自动启动服务

【常用操作】
- 启动: start.bat
- 停止: stop.bat
- 立即爬取并导出 Excel: crawl_now.bat
- 卸载自启服务: schtasks /delete /tn "PolicyMonitor" /f

【目录说明】
- python\          内置 Python 运行环境（请勿删除或移动）
- spiders\         爬虫模块
- templates\       网页模板
- static\          前端静态资源
- data\            SQLite 数据库存放位置
- output\          导出的 Excel 文件存放位置

【注意事项】
- 不要修改或删除 python\ 目录下的任何文件
- 整个文件夹可以整体复制/移动，路径可以任意（推荐避免中文路径）
- 防火墙需放行 5000 端口（局域网访问时）
- 服务器部署需要管理员权限
- 整体目录较大（500MB+），建议分发时打包为 zip
