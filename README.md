# 政策信息监控工具

公共事务部政策信息自动监控系统，爬取国家/省/市三级政府网站的政策通知，按关键词匹配过滤并提供 Web 管理界面。

## 快速使用（3步搞定）

> 前提：电脑需安装 Python 3.8+（安装时勾选 Add to PATH）

1. **双击 `install.bat`** — 自动安装所有依赖和浏览器引擎（首次使用，只需运行一次）
2. **双击 `start.bat`** — 启动应用
3. **打开浏览器访问 http://127.0.0.1:5000** — 开始使用

就这么简单！

## 功能特性

- 🌐 **15个政府网站自动爬取**（国家4个 + 四川5个 + 成都6个）
- 🔑 **关键词智能匹配** — 只保留匹配关键词库的文章
- 🎯 **选择性爬取** — 自由选择要爬取的网站
- 📅 **时间范围过滤** — 只抓取指定日期范围的内容
- 📊 **实时进度可视化** — 看到每个网站的爬取状态
- ⏸️ **暂停/取消** — 随时控制爬取任务
- 🚫 **自动去重** — 重复文章不会重复入库
- 📥 **导出Excel/CSV** — 支持按条件筛选后导出
- ✏️ **在线管理** — 关键词和网站配置都可在网页上修改
- 🔄 **定时爬取** — 每4小时自动执行一次
- 🛡️ **反爬绕过** — Playwright浏览器引擎 + 瑞数WAF对抗

## BAT 脚本说明

| 脚本 | 用途 |
|------|------|
| `install.bat` | 一键安装（创建环境 + 装依赖 + 下载浏览器） |
| `start.bat` | 启动 Web 应用 |
| `stop.bat` | 停止应用 |
| `crawl_now.bat` | 立即爬取并导出 Excel 到 output 目录 |
| `deploy_server.bat` | 部署到服务器（开机自启，需管理员权限） |

## 部署到 Windows 服务器

右键 `deploy_server.bat` → **以管理员身份运行**

部署后：
- 其他电脑通过 `http://服务器IP:5000` 访问
- 服务器重启自动启动服务
- 运行 `stop.bat` 停止服务

## 技术栈

- Python 3.8+ / Flask
- Playwright（无头浏览器爬取）
- SQLite（本地数据库）
- HTML/CSS/JS（前端界面）
- APScheduler（定时任务）

## 监控的网站

**国家级：** 科技部、工信部、卫健委、发改委
**四川省：** 发改委、经信厅、科技厅、卫健委、药监局
**成都市：** 发改委、高新区管委会、经信局、科技局、卫健委 + 高新通
