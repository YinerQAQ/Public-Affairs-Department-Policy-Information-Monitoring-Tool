# 系统架构图

本文档描述「公共事务部政策信息自动监控系统」的整体架构与组件关系。

## 1. 整体系统架构（分层视图）

系统采用经典分层架构，自顶向下分为：用户层、Web 层、业务层、数据层、外部数据源。

```mermaid
graph TB
    subgraph 用户层
        Browser[浏览器客户端<br/>Chrome / Edge]
    end
    subgraph Web层
        Flask[Flask Web服务<br/>端口5000]
        Static[静态资源<br/>HTML / CSS / JS]
    end
    subgraph 业务层
        Scheduler[APScheduler<br/>定时调度器<br/>每4小时]
        SpiderEngine[爬虫引擎<br/>Playwright 无头浏览器]
        KeywordFilter[关键词过滤器<br/>64个关键词]
        Exporter[数据导出模块<br/>Excel / CSV]
        AntiWAF[瑞数WAF反检测]
    end
    subgraph 数据层
        MySQL[(MySQL 数据库<br/>articles / keywords / websites)]
    end
    subgraph 外部数据源
        National[国家级网站 x4<br/>国务院 / 卫健委 / 药监局 / 医保局]
        Province[省级网站 x5<br/>四川省政府等]
        City[市级网站 x6<br/>成都市 / 高新区等]
    end

    Browser --> Flask
    Flask --> Static
    Flask --> Scheduler
    Flask --> SpiderEngine
    Flask --> Exporter
    Scheduler --> SpiderEngine
    SpiderEngine --> AntiWAF
    SpiderEngine --> KeywordFilter
    KeywordFilter --> MySQL
    Exporter --> MySQL
    SpiderEngine --> National
    SpiderEngine --> Province
    SpiderEngine --> City
```

## 2. 组件关系图

展示各核心模块之间的调用、依赖与数据流向。

```mermaid
graph LR
    User[用户] -->|HTTP请求| AppPy[app.py<br/>Flask 路由]
    AppPy -->|读取配置| ConfigPy[config.py<br/>配置管理]
    AppPy -->|触发爬取| SchedulerPy[scheduler.py<br/>调度器]
    AppPy -->|渲染页面| Templates[templates/<br/>Jinja2模板]
    AppPy -->|CRUD| DatabasePy[database.py<br/>数据访问层]

    SchedulerPy -->|调用| CrawlExport[crawl_export.py<br/>爬取与导出主流程]
    CrawlExport -->|实例化| BaseSpider[spiders/base.py<br/>BaseSpider基类]

    BaseSpider -->|继承| NationalSp[national.py<br/>国家级爬虫]
    BaseSpider -->|继承| SichuanSp[sichuan.py<br/>四川省爬虫]
    BaseSpider -->|继承| ChengduSp[chengdu.py<br/>成都市爬虫]
    BaseSpider -->|继承| GaoxinSp[gaoxintong.py<br/>高新区爬虫]

    BaseSpider -->|调用| Playwright[Playwright<br/>Chromium]
    BaseSpider -->|过滤| KeywordFilter[关键词匹配]
    BaseSpider -->|去重落库| DatabasePy

    DatabasePy -->|连接| MySQL[(MySQL)]
```

## 3. 架构说明

| 层级 | 组件 | 职责 |
| --- | --- | --- |
| 用户层 | 浏览器 | 操作 Web 管理界面，查看进度、结果与导出 |
| Web 层 | Flask + 模板 | 提供路由、渲染、API 接口 |
| 业务层 | 调度器 / 爬虫引擎 / 过滤器 / 导出器 | 实现自动爬取、去重、过滤、导出 |
| 数据层 | MySQL | 持久化文章、关键词、网站配置 |
| 外部数据源 | 15 个政府站点 | 国家级 4 个、省级 5 个、市级 6 个 |
