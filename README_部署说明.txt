========================================
政策信息监控工具 - 部署说明
========================================

【本机使用（给同事）】
1. 把整个文件夹复制到目标电脑
2. 双击 install.bat 安装（只需一次）
3. 双击 start.bat 启动
4. 打开浏览器访问 http://127.0.0.1:5000

【服务器部署（后台运行+开机自启）】
1. 把文件夹复制到服务器
2. 右键 deploy_server.bat → 以管理员身份运行
3. 部署完成后，其他电脑通过 http://服务器IP:5000 访问
4. 服务器重启后会自动启动服务

【常用操作】
- 启动: start.bat
- 停止: stop.bat
- 立即爬取并导出Excel: crawl_now.bat
- 卸载自启服务: schtasks /delete /tn "PolicyMonitor" /f

【注意事项】
- 需要 Python 3.8+ （安装时勾选 Add to PATH）
- 首次安装需联网（下载依赖和浏览器引擎）
- 服务器部署需要管理员权限
- 防火墙需放行 5000 端口
